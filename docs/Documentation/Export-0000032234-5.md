# Export

### Export

Georeferenced images can be exported in PNG, JPG, TIFF and geoTIFF format.

![Image](./data/icons/note.gif)

Georeferenced images from a project can be exported to be used in other projects.

From the properties, select the    Export to Global option.

![Image](graphics/00466037.jpg)

**Export to Global**

| 1. | Select the image to export. |
| --- | --- |
| 2. | Select Georeference Image. |
| 3. | Edit the suggested file name. |
| 4. | Define the world file units. |
| 5. | Navigate to the desired export location. |
| 6. | Select Export. |
